Excel File Standardizer - New Index

1. Sheet Selection Dropdown

The sheet selection dropdown is now always visible when an Excel file has sheets.

2. Topics Field Not Mandatory

The Topics field is no longer mandatory, allowing you to leave it unmapped or blank.

3. Simplified File Splitting Text

The text for file splitting has been simplified to just "Enable File Splitting".

4. iMocha Branding

Text now refers to "iMocha's standard template format" instead of "our standard template format".
